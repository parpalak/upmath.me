What's new

2024-05-08 Fixed some inconsistencies in sync scroll

During text editing and appending at the very end, the preview area did not automatically scroll to the bottom, causing inconvenience when entering certain formulas and images.

2024-04-28 Multiple Document Support and Version History

This update has some features that enhances your document editing experience and prevents any unfortunate incidents of text loss. Now, you can work with multiple documents and restore old text versions.

New features:

  • Main menu accessible via the ☰ button in the header.
  • New documents can be created from the "☰ → New" menu option.
  • Switching between several documents is possible via the "☰ → Open" menu option.
  • Version Recovery: Restore previous document versions with the "☰ → Restore from history."
  • Download and Upload: These functions have been moved to the menu.
  • Full-Screen: Toggle to full-screen mode through a "☰ → Enter full-screen" menu item. Perfect for tablet users, as it maximizes workspace by eliminating mobile browser navigation elements.

2024-02-12 Download HTML with built-in SVG equations

Now you can download HTML with the built-in SVG equations from the "preview" tab. See discussion on GitHub.

2024-01-10 Added "esvect" package

Added "esvect" package support for nice vector arrows: \vv{AB} → $$\vv{AB}$$.

2023-05-11 Korean language support

Now the "kotex" package is automatically enabled if you write Korean characters: \text{안} → $$\text{안}$$.

2022-07-03 Added "mhchem" package

You can write chemical equations with the "mhchem" package. Example:

\ce{Hg^2+ ->[I-] HgI2 ->[I-] [Hg^{II}I4]^2-}

$$\ce{Hg^2+ ->[I-] HgI2 ->[I-] [Hg^{II}I4]^2-}}$$.

2020-07-26 Implemented styles for printing.

You can use the browser print window to print the document or save it as PDF.

2019-12-22 Added "xcolor" package

It allows to use different colors in the LaTeX text. Example:

\pagecolor{yellow!20!white}
\begin{minipage}{10cm}
\begin{enumerate}
\color{orange!61!black}
  \item List entries with a color.
  \item List numbers are colored too.
\end{enumerate}

The background color of text can also be
\textcolor{red}{easily} set.\\
For instance, you can change use a
\colorbox{green!20!white}{green background}
and then continue typing.
\end{minipage}

$$\pagecolor{yellow!20!white} \begin{minipage}{10cm} \begin{enumerate} \color{orange!61!black} \item List entries with a color. \item List numbers are colored too. \end{enumerate} The background color of text can also be \textcolor{red}{easily} set.\\ For instance, you can change use a \colorbox{green!20!white}{green background} and then continue typing. \end{minipage}$$

2018-05-02 Support for some Unicode characters

Several Unicode characters (like →, ⇒, ⊕, Σ, ...) can now be used directly. See discussion on GitHub.

2018-03-04 Added "circuitikz" package

It allows to draw circuit diagrams:

\begin{circuitikz}
\draw
(0,0) node[above] {$A$} to[short, o-*] ++(0.7,0)
coordinate (A) to[generic, *-*] ++(2,0)
coordinate (B) to[generic, *-*] ++(2,0)
coordinate (E) to[generic, *-*] ++(2,0)
coordinate (D) to[short, *-o] ++(0.7,0) node[above] {$B$};
\draw (A)-- ++(0,-0.7)-| (E) (B)-- ++(0,0.7)-| (D);
\end{circuitikz}

2018-01-03 Added "tikz-3dplot" package

It allows to draw 3D plots:

\tdplotsetmaincoords{70}{10}
%
\pgfmathsetmacro{\r}{0.8}
\pgfmathsetmacro{\ra}{.45}
\pgfmathsetmacro{\rb}{.5}
\pgfmathsetmacro{\thetavec}{70}
\pgfmathsetmacro{\phivec}{20}
\pgfmathsetmacro{\a}{20}
\pgfmathsetmacro{\cms}{3}
%
\begin{tikzpicture}[scale=5,tdplot_main_coords]
\coordinate (O) at (0,0,0);
\draw[->] (-1,0,0) -- (1,0,0) node[anchor=north east]{$x$};
\draw[->] (0,-1.2,0) -- (0,1.2,0) node[anchor=west]{$y$};
\draw[->] (0,0,-0.6) -- (0,0,0.6) node[anchor=west]{$z$};
\tdplotsetcoord{P}{\ra}{90-\a}{\phivec}
\tdplotsetcoord{G}{\ra}{90-\a}{\phivec+10}
\tdplotsetcoord{R}{\rb}{90-\a}{\phivec}
\tdplotsetcoord{T}{\rb}{90-\a}{\phivec+10}
\draw[dashed] (P) -- (Pxy) (G) -- (Gxy) (T) -- (Txy) (R) -- (Rxy) node[pos=0.4,right] {$h$};
\draw[fill=red,fill opacity=0.6] (P) -- (G) -- (T) -- (R) -- (P);
\draw[fill=orange,fill opacity=0.7] (Pxy) -- (Gxy) -- (Txy) -- (Rxy) -- cycle;
\draw[dashed] (Gy) -- (Gxy) node[pos=0.5,anchor=north,inner sep=2pt] {$x$};
\draw[] (Gy) -- (G);
%
\tdplotsetrotatedcoords{90}{-90}{-90}
\tdplotdrawarc[tdplot_rotated_coords]{(Gy)}{0.2}{0}{22}{right}{$\alpha$}
\tdplotsetrotatedcoords{0}{-\a}{0}
\draw[red,dashed,tdplot_rotated_coords,fill=red,fill opacity=0.1] (0,\r,0) arc (90-\cms:450-\cms:\r);
\draw[red,thick,tdplot_rotated_coords] (0,\r,0) arc (90-\cms:-200:\r);
\draw[orange,dashed,fill=orange,fill opacity=0.1] (0,\r,0) arc (90+\cms:450+\cms:\r);
\draw[orange,thick] (0,\r,0) arc (90+\cms:408:\r);
\end{tikzpicture}

2017-12-20 Added "stmaryrd" as a default package

This is the new St Mary’s Road symbol font. It provides a number of new symbols, including ones for derivation of functional programming, process algebra, domain theory, multisets and many more. See discussion on GitHub.

2017-10-31 Added "bussproofs.sty" package

Sam Buss’s powerful and flexible package bussproofs.sty offers macros for setting natural deduction and sequent proofs.
